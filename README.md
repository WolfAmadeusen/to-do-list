# Todo List на Python

## Возможности программы

### Задача 1: Добавить задачу

Пользователь может добавить новую задачу в список.

### Задача 2: Отметить задачу выполненной

Можно пометить любую задачу как выполненную.

### Задача 3: Удалить задачу

Позволяет удалить задачу по её `id`.

## Как запустить программу

4. Запустите программу командой:

```bash
python main.py
```
